================================================================================
ICONES E IMAGENS NECESSARIOS PARA O SISTEMA
================================================================================

Este arquivo lista os icones (.ico) e imagens necessarios para o sistema
funcionar corretamente. Copie os arquivos para esta pasta (C:\4c\icons\).

LOGO PARA RELATORIOS (OBRIGATORIO):
-----------------------------------
- logo.bmp       (Logo da empresa para cabecalho dos relatorios)
                 Tamanho recomendado: 200x50 pixels
                 Formato: BMP (bitmap)

ICONES PARA BOTOES DE LISTA (tela principal dos cadastros):
-----------------------------------------------------------
- Add.ico        (Incluir novo registro)
- Book.ico       (Visualizar/Consultar registro)
- altera.ico     (Alterar registro)
- Delete.ico     (Excluir registro)
- Search.ico     (Buscar/Pesquisar)
- Shutdown.ico   (Encerrar/Fechar formulario)

ICONES PARA BOTOES DE EDICAO (tela de dados):
---------------------------------------------
- button_ok.ico      (Confirmar/Salvar)
- button_cancel.ico  (Cancelar)

ICONES PARA GRIDS (selecao em listas):
--------------------------------------
- geral_selecionar_26.jpg   (Selecionar item no grid)
- cadastro_excluir_26.jpg   (Excluir item do grid)

ICONES PARA RELATORIOS:
-----------------------
- relatorio_video_26.jpg      (Visualizar relatorio)
- relatorio_impressora_26.jpg (Imprimir relatorio)
- geral_envelope_32.jpg       (Enviar por email)
- relatorio_sair_60.jpg       (Fechar relatorio)

NOTAS:
------
- Se os icones nao existirem, os botoes serao exibidos apenas com texto
- O logo.bmp eh referenciado pela variavel global gc_4c_LogoRelatorio
- Os icones sao carregados de gc_4c_CaminhoIcones (caminho relativo)
- Formatos suportados: .ico, .bmp, .jpg
